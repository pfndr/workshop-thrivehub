## What this is

A 3-hour hands-on workshop that teaches one durable skill: a loop you can run on any idea, now or 10 years from now. Attendees write a falsifiable hypothesis, map a customer journey on paper, build a rough prototype, and test it on real humans in the room across three rapid loops (Idea, Plan, Create, Test). In the final hour, everyone runs the same loop on their own idea inside ProtoVibing, an AI-powered founder tool, then interviews the room and feeds the learnings back for synthesis. No code, no pitch decks, no pressure to "have a startup."

## Running the deck

Open `index.html` in a browser on a laptop, desktop, or tablet in landscape. Arrow keys or space to navigate, F for fullscreen, Esc to exit. Timer slides play LoFi backing music (HoliznaCC0, CC0 1.0 Universal, freemusicarchive.org); music pauses when a timer pauses.
